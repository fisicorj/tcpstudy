Cable Trainer Pro
A modern and interactive web application designed to help students and professionals master the art of Ethernet cable termination. Practice the TIA/EIA 568A, 568B, and advanced Crossover standards with a user-friendly drag-and-drop interface, real-time validation, and a helpful tutorial mode.

Features
Interactive Drag-and-Drop: Visually assemble virtual Ethernet cables by dragging and dropping wires into an RJ45 connector.

Multiple Wiring Standards: Practice essential standards, including TIA/EIA 568A, 568B, and both 10/100 and 1000BASE-T Crossover.

Real-Time Validation: Get instant visual feedback as you place each wire, with slots turning green for correct placement and red for incorrect.

Tutorial Mode: A guided, step-by-step mode that highlights the correct wire and slot, perfect for beginners learning the color codes.

Responsive Design: The application is fully responsive and works seamlessly on desktop, tablet, and mobile devices.

Clean and Modern UI: A sleek, minimal design with smooth animations and a pleasant user experience.

How to Use
Select a Pattern: Choose the desired wiring standard from the dropdown menu (e.g., 568A, 568B).

Drag and Drop: Click and drag the colored wires from the pool at the top into the corresponding slots in the RJ45 connector.

Real-Time Feedback: Observe the slots. They will light up in green if the wire is correct or red if it is incorrect.

Tutorial Mode: If you need help, click the "Modo Tutorial" button. The correct wire and slot for the next step will be highlighted. You can exit this mode at any time.

Restart: Click the "Reiniciar" button to reset the current challenge and start over with a new, randomized set of wires.

Technologies Used
HTML5: For the core structure of the application.

CSS3 & Tailwind CSS: For styling, animations, and a responsive layout.

JavaScript (ES6+): For all the interactive logic, including drag-and-drop, validation, and game state management.

Font Awesome: For icons used in the UI.
